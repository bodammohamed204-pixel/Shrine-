# Shrine Privacy Policy Draft

Last updated: June 27, 2026

Shrine lets users create, search, and manage memorial profiles. This draft must be reviewed, customized, and hosted on a public website before submitting the app to Google Play or the Apple App Store.

## Information We Collect

Shrine may collect account information such as name, email address, country, mobile number, and WhatsApp OTP verification status. Users may add memorial information such as names, dates, country, photos, and notes.

## How We Use Information

We use information to create accounts, verify access through WhatsApp OTP, save memorial entries, provide search and browsing features, and support user requests.

## Third-Party Services

Shrine uses Oncallos to send and verify WhatsApp one-time passwords. Phone numbers are sent to Oncallos only for OTP delivery and verification.

## Data Storage

The current app stores account and memorial data locally in the user's browser or mobile webview storage unless a production backend is added. Clearing app data can remove locally stored content.

## User Choices

Users can edit profile details, create memorial entries, block entries, and log out from the app. For deletion or support requests, users should contact the app owner through the support channel listed in the app.

## Security

We use server-side handling for OTP API secrets. API keys must never be shipped inside the mobile app.
